# Random Encounter Tables
## DL1: Dragons of Despair — Rules Engine Reference

---

## TABLE 1: Encounter Type and Frequency

Random encounters occur on a **roll of 1 on 1d10**, at the frequency noted below according to terrain. If an encounter is indicated, determine distance according to the table (1 = 10 yards), then roll to generate a number in the Monster Range for that terrain. Use Table 2 for all encounters except the Darken Wood; use Table 3 for Darken Wood.

| Terrain | Check Frequency | Distance Roll | Monster Range |
|---------|----------------|--------------|--------------|
| **Towns** | Once every 3 turns | 1d6 | 1–6 (1d6) |
| **Plains** | 4 times per day | 1d100 | 3–14 (1d12+2) |
| **Forest** | 6 times per day | 1d10 | 4–15 (1d12+3) |
| **Hills/Mountains** | 4 times per day | 1d20 | 7–18 (1d12+6) |
| **Marsh** | 8 times per day | 1d4 | 13–24 (1d12+12) |
| **Ruins** | Once every 3 turns | 1d6 | 11–22 (1d12+10) |

**Note:** Use Table 3 for all encounters in the **Darken Wood** (areas 22–26). The Darken Wood has a special encounter frequency — see Table 3.

## Encounter Area Summary

| Terrain Type | Random Encounters Include |
|-------------|--------------------------|
| Towns | Townspeople, disguised draconians, Seeker Guards |
| Plains | Plainsmen, elves, wolves, draconians, trolls, ogres |
| Forest | Satyrs, sprites, spiders, draconians, will-o-wisps |
| Hills/Mountains | Draconians, trolls, giants (if in range), wraiths, will-o-wisps |
| Marsh | Snakes, catoblepas, hatchling dragons, water creatures (from area 44b rules) |
| Ruins | Draconians (all types), wraiths, undead, spectral minions |
| Darken Wood | Forest creatures; guided to Forestmaster unless PCs are enemies |
