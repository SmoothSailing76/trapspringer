# Known Limitations

Trapspringer v0.2 Complete is a functional main-path DL1 prototype, not a full replacement for a human AD&D referee.

## DL1 content limitations

- The canonical main path is implemented from Event 1 to Epilogue.
- Optional wilderness branches, many side encounters, and alternate failure paths are not exhaustive.
- Some main-path scenes are represented as deterministic spine events rather than fully open keyed-room play.

## AD&D 1e rules limitations

- Combat uses compact procedures rather than full segment-by-segment AD&D timing.
- The spell system covers main-path spells only.
- Monster special abilities are partial.
- XP, training, hirelings, item saves, and downtime are not complete.

## Party simulation limitations

- The simulated table has distinct voices and tendencies, but long-term human-like memory and evolving player habits are still limited.

## Mapping and knowledge limitations

- Public/private boundaries are enforced for main-path secrets, but the map system is not yet a full dungeon cartography engine.

## Replay limitations

- Replay is designed around logged state diffs and named checkpoints for the main path. Arbitrary open-ended future branches may need additional replay tests.
