# Install and Run

From project root:

```bash
python -S -m trapspringer.adapters.cli.main --campaign dl1 --demo-main-path --no-recap
```

Useful:

```bash
python -S -m trapspringer.adapters.cli.main --status
python -S -m trapspringer.adapters.cli.main --list-saves
python tools/verify_release.py
python tools/verify_replay_spine.py
```
