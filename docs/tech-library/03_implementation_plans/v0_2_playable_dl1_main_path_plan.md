# v0.2.0 Playable DL1 Main Path Plan

The v0.2 target is a complete playable DL1 main path from Event 1 to Epilogue with save/load and deterministic replay.

## Main Milestones

Event 1, Goldmoon joins, Xak Tsaroth arrival, 44f, 44k, Temple of Mishakal, staff recharge, descent, lower city, secret route, 70k, staff shatter, collapse escape, epilogue.

## Required Checkpoints

start_of_event_1, after_event_1, goldmoon_joined, arrival_xak_tsaroth, after_44f, after_44k_surface, temple_mishakal, staff_recharged, descent_started, lower_city, secret_route_known, before_70k, staff_shattered, collapse_escape, epilogue_complete.
